### open telecom platform  (OTP)  是什麼
用來開發 erlang 的應用程式的一個平台，可以讓開發者建立一個用來開發分布式、高可用的應用程式。


### OTP 包含哪些項目

1. erlang 的直譯器
2. erlang 的編譯器
3. 通訊協定
4. [Corba Object Request Broker]
5. static analysis tool called Dialyzer
6. [Mnesia] 一個資料庫
7. 一些函式庫
