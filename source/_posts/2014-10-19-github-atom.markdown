[Atom] 是 Github 基於 node.js + webkit 開發強調模組化的一款編輯器，看過源碼可以知道是使用 CoffeeScript 和 LESS 撰寫的，
因此如果是 web 的開發者也能修改界面，裡面也集成了一些還蠻實用的功能，網路上有人說會有些不順的感覺不過我下載執行感覺起來還蠻順的，
有些工作上要使用的 Package 也有人開發，之後如果使用順利的話會從 ST 換成 Atom。




[Atom]:https://atom.io/
